## 🚀 Quick Start | 快速开始 | クイックスタート

```bash
# 🌟 Install dependencies | 安装依赖 | 依存関係をインストール
bun install

# 🎬 Start development server | 启动开发服务器 | 開発サーバーを起動
bun dev

# 🗄️ Run database migrations | 运行数据库迁移 | データベースマイグレーションを実行
bun db:migrate

# 🐳 Start Docker Compose (PostgreSQL + Redis) | 启动 Docker Compose | Docker Compose を起動
docker-compose up -d

# 🎯 Build for production | 构建生产版本 | 本番ビルド
bun build

# 🔍 Check code quality | 检查代码质量 | コード品質をチェック
bun check
```

```javascript
// 🎯 Connect to real-time collaboration
// 连接实时协作 | リアルタイムコラボレーションに接続
const ws = new WebSocket('wss://api.my-aistack.com/ws');

// 🤝 Join collaboration session
// 加入协作会话 | コラボセッションに参加
ws.send(JSON.stringify({
  type: 'join_session',
  sessionId: 'sess_123'
}));

// 📥 Listen for agent messages
// 监听智能体消息 | エージェントメッセージを受信
ws.onmessage = (event) => {
  const data = JSON.parse(event.data);
  console.log(`🤖 ${data.agentId}:`, data.message);
};
```

### Original Features | 原始功能 | 元々の機能

- **TypeScript** - For type safety | 类型安全 | 型安全

- **Better-Auth** - Authentication | 认证 | 認証
- **Turborepo** - Optimized monorepo | 最適化モノレポ | 最適化モノレポ

### Original Scripts | 原始脚本 | 元々のスクリプト
